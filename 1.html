  1. HTML이란?

    • 내가 생각하는 HTML

      • internet을 통해 내가 원하는 자료를 web을 통해 볼 수 있도록 해주는 프로그램?이다. HTML만을 사용하였을 경우, 원하는 content를 글로만 표현할 수 있는 한계가 있다. 다양한 이미지로 꾸미고 싶다면, javascript나, css와 함께 버무려져야한다.

    • HTML의 사전 정의

      • HTML 은 Hyper Text Markup Language 약어로 HyperText(웹 페이지에서 다른 페이지로 이동할 수 있도록 하는 것) 기능을 가진 문서를 만드는 언어입니다. 다시 말해, 구조를 설계할 때 사용되는 언어로 hyper link 시스템을 가지고 있으며, 흔히 말하는 웹 페이지를 위한 마크업 언어라고 할 수 있습니다. 앞으로 공부할 CSS 가 비주얼 디자인(Visual Design) 이라면 HTML 은 구조적 설계(Structure Design) 이라 할 수 있습니다.
